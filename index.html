Get started!

Open up index.html start to write HTML code, and see the magic happen! As you can see, it is not needed to write the majority of CSS code, since CometCSS targets HTML elements by thier name and syles them for you so you can focus on more important stuff! 

$ git clone https://github.com/brandinos/CometCSS.git
$ cd cometCSS
$ npm install
$ npm run sass
